# VS Code Keyboard Shortcuts — Mac & Windows

A handy cheat sheet of the most-used VS Code shortcuts for everyday coding and testing.

**Key symbols (Mac):** `⌘` Command · `⌥` Option/Alt · `⌃` Control · `⇧` Shift · `↩` Enter

---

## 1. Essentials

| Action                         | Mac              | Windows / Linux      |
| ------------------------------ | ---------------- | -------------------- |
| Command Palette (do anything)  | `⌘ ⇧ P`          | `Ctrl Shift P`       |
| Quick Open file                | `⌘ P`            | `Ctrl P`             |
| Open Settings                  | `⌘ ,`            | `Ctrl ,`             |
| Toggle Terminal                | `` ⌃ ` ``        | `` Ctrl ` ``         |
| New File                       | `⌘ N`            | `Ctrl N`             |
| Save                           | `⌘ S`            | `Ctrl S`             |
| Save All                       | `⌘ ⌥ S`          | `Ctrl K S`           |
| Close File                     | `⌘ W`            | `Ctrl W`             |

## 2. Editing

| Action                          | Mac              | Windows / Linux      |
| ------------------------------- | ---------------- | -------------------- |
| Cut line (no selection)         | `⌘ X`            | `Ctrl X`             |
| Copy line (no selection)        | `⌘ C`            | `Ctrl C`             |
| Move line up / down             | `⌥ ↑` / `⌥ ↓`    | `Alt ↑` / `Alt ↓`    |
| Copy line up / down             | `⌥ ⇧ ↑` / `⌥ ⇧ ↓`| `Shift Alt ↑` / `↓`  |
| Delete line                     | `⌘ ⇧ K`          | `Ctrl Shift K`       |
| Insert line below / above       | `⌘ ↩` / `⌘ ⇧ ↩`  | `Ctrl ↩` / `Ctrl Shift ↩` |
| Indent / Outdent                | `⌘ ]` / `⌘ [`    | `Ctrl ]` / `Ctrl [`  |
| Comment / uncomment line        | `⌘ /`            | `Ctrl /`             |
| Block comment `/* */`           | `⌥ ⇧ A`          | `Shift Alt A`        |
| Undo / Redo                     | `⌘ Z` / `⌘ ⇧ Z`  | `Ctrl Z` / `Ctrl Y`  |
| Format document                 | `⌥ ⇧ F`          | `Shift Alt F`        |

## 3. Selection & Multi-Cursor

| Action                              | Mac              | Windows / Linux      |
| ----------------------------------- | ---------------- | -------------------- |
| Select current line                 | `⌘ L`            | `Ctrl L`             |
| Select all                          | `⌘ A`            | `Ctrl A`             |
| Add cursor above / below            | `⌘ ⌥ ↑` / `⌘ ⌥ ↓`| `Ctrl Alt ↑` / `↓`   |
| Add cursor at click                 | `⌥ Click`        | `Alt Click`          |
| Select next match (multi-edit)      | `⌘ D`            | `Ctrl D`             |
| Select ALL matches of word          | `⌘ ⇧ L`          | `Ctrl Shift L`       |
| Expand / shrink selection           | `⌃ ⇧ →` / `←`    | `Shift Alt →` / `←`  |

## 4. Navigation

| Action                          | Mac              | Windows / Linux      |
| ------------------------------- | ---------------- | -------------------- |
| Go to Line                      | `⌃ G`            | `Ctrl G`             |

| Next / Previous tab             | `⌘ ⌥ →` / `←`    | `Ctrl PageDn` / `Up` |

## 5. Search & Replace

| Action                          | Mac              | Windows / Linux      |
| ------------------------------- | ---------------- | -------------------- |
| Find (in file)                  | `⌘ F`            | `Ctrl F`             |

| `F5`                        | Run / debug                           |

> 💡 **Tip:** Forgot a shortcut? Press `⌘ ⇧ P` (Mac) / `Ctrl Shift P` (Windows) and type the
> action name — the shortcut is shown right next to it. You can also open the full list with
> **Keyboard Shortcuts** (`⌘ K ⌘ S` / `Ctrl K Ctrl S`).
